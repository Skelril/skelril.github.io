Set up mumble

  1. Download the Mumble Client [Click me!]
  2. Install the program, as you would any other
  3. Start mumble
  4. Go through the audio configuration wizard
    • If not prompted you may find it under Configure > Audio Wizard
  5. Under the Server menu click Connect
  6. Click Add New
  7. Enter the details of your prefered Skelril mumble server

Then...

  1. Go under Server > Connect
  2. Pick Skelril from the list
  3. Enjoy
